# My Automated Testing Ecosystem Hub

Welcome to my centralized automation portfolio. This repository contains various testing frameworks and scripts covering UI, API, and Performance testing.

##  Repository Architecture

* **`UI-Automation/`**
  * `Playwright-Python/`: Web automation using Playwright and Python.
  * `Selenium-Java/`: UI testing using Selenium WebDriver and Java (Eclipse ecosystem).
  * `Selenium-Python/`: UI testing using Selenium WebDriver and Python (PyCharm ecosystem).
  * `SQL-Gherkin/`: BDD testing using Cucumber/Gherkin syntax alongside SQL validations.

* **`API-Testing/`**
  * `RestAssured-Java/`: REST API automation using the RestAssured library in Java.

* **`Load-Testing/`**
  * `Apache-JMeter/`: Performance and load testing scripts utilizing Apache JMeter.

## Tools & Technologies Used
* **Languages:** Java, Python, SQL
* **Frameworks:** Selenium, Playwright, RestAssured, Cucumber (Gherkin)
* **Performance:** Apache JMeter
* **IDEs:** Eclipse, PyCharm
